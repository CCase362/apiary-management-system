# Sunny Day Apiary Management System

A full-stack web application designed to help manage apiary operations including hive tracking, inspections, and inventory management.

## Technologies Used

- Java
- JSP
- CSS
- SQL

## Features

- Hive tracking and management
- Inspection logging
- Inventory organization
- Database-backed data storage
- Responsive user interface design

## Project Overview

This project was developed as part of a university database systems project focused on designing and implementing a management system for apiary operations. The application combines front-end user interaction with back-end database functionality to organize and manage hive-related data.

## What I Learned

- Full-stack application structure
- Database integration and SQL workflows
- Front-end and back-end interaction
- Organizing multi-file Java web applications
- UI/UX-focused design principles
